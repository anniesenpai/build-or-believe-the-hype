# 🌸 Build or Believe the Hype?

An interactive branching narrative built with Twine (Harlowe) for a Digital Games & Learning course at UBC.

## About

You play as an instructional designer pitching a computer game-making program to a middle school. Every choice you make — who to design for, how to teach, what outcomes to target — is tested against what the research actually shows.

The game is based on two readings:

- **Burke, Q. & Kafai, Y. (2014).** A decade of game making for learning: From tools to communities. *Handbook of Digital Games.*
- **Denner, J., Campe, S., & Werner, L. (2019).** Does computer game design and programming benefit children? A meta-synthesis of research. *ACM Transactions on Computing Education, 19*(3).

## Features

- 🌸 Japanese school-themed UI
- 📚 Evidence-based branching — choices reflect real research findings
- 🎵 Optional lofi background music
- 承認 / 不合格 stamp animations on the endings
- Built with Twine 2 + Harlowe 3
